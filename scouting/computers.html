Computer Resources

This is a place for links to software, everything that you should need to get your computer up and running the software needed to do the robot good, as well as information about the current team computers. In the future we hope to be able to host some of the files locally so that reinstallation can happen at tournaments without internet connection. For now, it is just a bunch of links to installation and download instructions.
